How it works

Plain English

itsgiving uses data from Camfess and Oxfess to train a machine learning model to understand the semantic `vibes' of each college. When users input a new word, the machine learning model returns the top 3 colleges most associated with the vibe of that word.

Technical Details

We use the Word2Vec vector embedding model of words. We generate a vector within this embedding space for each college, by using a weighted average over all words in the Camfess/Oxfess posts associated with that college (with weights corresponding to a TF-IDF measure of the significance of each word). When a user inputs a new word, we simply return the three top colleges whose vectors have the largest cosine similarities with the input word.

(Disclaimer: itsgiving is not affiliated with any university or institution and does not support any of the associations made by the ML model.)
